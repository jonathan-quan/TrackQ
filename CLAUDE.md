# Parlay Tracker

## What It Does
NBA parlay tracking dashboard. Users add player prop bets, app fetches real stats and marks each leg hit or miss automatically.

## Tech Stack
- **Frontend:** React + Tailwind CSS
- **Backend:** FastAPI
- **Database:** MySQL via SQLAlchemy (PyMySQL driver)
- **NBA Data:** `nba_api` (free, no key needed)
- **Auth:** JWT + bcrypt

## Current Phase
- NBA only
